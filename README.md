# PSICONR VISION

**Sistema Completo de Gestão de Riscos Psicossociais – NR-01**

Aplicação web para realizar diagnóstico, matriz de risco, plano de ação e monitoramento contínuo de riscos psicossociais conforme a nova NR-01 (vigência 26/05/2026).

### Funcionalidades
- Diagnóstico automatizado (COPSOQ adaptado)
- Matriz de risco automática
- Plano de ação gerado por IA
- Dashboard com gráficos profissionais
- Evolução temporal dos riscos
- Exportação PDF e Excel
- Histórico de diagnósticos por empresa

**Tecnologia:** Python + Streamlit

**Desenvolvido por:** Emanuelle Melo

---

## Como usar

1. Faça login com seu e-mail
2. Faça upload do arquivo de respostas do questionário
3. O sistema gera automaticamente: diagnóstico, matriz, plano de ação e gráficos
